Requirements
================

JDK : `1.8+`  
Maven : `3.5+`

IDE Requirements
================

Lombok support plugin and annotation processing required.

How to build
================
`mvn clean install` or `mvn clean test`

Code locations
================

Task 1 : see  `VersionSorter` and `VersionSorterTest` classes.   
Task 2 : see  `StationTask` and `StationTaskTest` classes.
